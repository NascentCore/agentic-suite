# Template Assets

This directory stores reusable markdown template assets for the universal OpenClaw-like scaffold product.

Files:
- `TEMPLATE_SOUL.md`
- `TEMPLATE_SKILLS.md`
- `TEMPLATE_AGENTS.md`
- `TEMPLATE_TOOLS.md`
- `TEMPLATE_SKILL_ALIAS.md`
- `TEMPLATE_STYLE.md`

These files mirror the generated output structure and can be reused by external renderers or future non-Python implementations.
